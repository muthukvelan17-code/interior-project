# Feature Roadmap: InteriorConnect Tamil Nadu

## Phase 1: MVP Core (Months 1-2)
**Goal:** Prove the concept, acquire early designers, and enable basic matchmaking.
- Authentication (Email + JWT)
- Client and Designer Onboarding Roles
- Designer Profiles (Bio, Location, Basic Portfolio Upload)
- Search & Discovery (Filter strictly by Location string & Styles)
- Basic Messaging (In-app text chat using Socket.io to negotiate quotes)
- Static Landing Page (Tailwind/Vanilla CSS matching premium aesthetics)

## Phase 2: Booking & Monetization (Months 3-4)
**Goal:** Enable platform transactions and revenue generation.
- Service Packages (Designers can list price packages)
- Project Booking Engine (Request Quote -> Accept -> Milestone Creation)
- Payment Gateway Integration (Razorpay for upfront consultation fee / milestone payments)
- Platform Commission System (Automated deduction)
- User Reviews & Ratings

## Phase 3: Complete Designer Workspace (SaaS Features) (Months 5-6)
**Goal:** Increase platform stickiness by providing comprehensive business tools for designers.
- Designer Project Management Dashboard (Kanban boards, Timelines)
- Lead & Client Management (CRM) for tracking inquiries and proposals
- File Sharing & Client Approval workflows (Mood boards, revisions)
- Designer Business Tools: Invoice generation, Contract templates, Expense tracker
- Virtual Design Packages specifically (Digital Delivery capabilities)

## Phase 4: Procurement, Growth, & Advanced AI (Months 7-8)
**Goal:** Additional revenue streams and state-of-the-art AI UX.
- Curated Shopping Lists & Procurement Planner (Vendor integration)
- Designer Membership Tiers (Pro tier gets premium placement & analytics dashboard)
- Designer Growth Tools (Boost profile, analytics for leads/views)
- AI Mood Board Generator & AI Cost Estimation Assistant
- AI-based designer recommendation system for clients

## Phase 5: Future & "Bonus" Features (Beyond)
- AR Room Visualization Preview for certain materials (WebXR)
- Tamil + English Multilingual toggle across the platform
- Video Consultation Scheduling (Integration with Zoom/Agora APIs)
